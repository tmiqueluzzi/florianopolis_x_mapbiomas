## Pasta para dados processados

1. florianopolis1985 é um exemplo da tentativa de simplesmente cortar o raster original com a máscara
2. florianopolis1985_v1 é a mesma imagem, porém a máscara contendo o buffer de 50m;
3. 1985_31982 é a imagem anterior reprojetada para Sirgas 2000 UTM 22 S
4. 'Diferença nas áreas' é um csv contendo a diferença entre a soma das áreas das feições de cada distrito versus a suposta área de cada distrito. Maior diferença percentual ficou em menos de 0.09%
5. Uso do Solo em Coqueiros 1996 é um exemplo de imagem de saída, contendo o uso do solo, uma legenda e uma tabela contendo a porcentagem de cada classe. Foram feitos para todos os anos entre 1985 e 2024 para cada distrito do município
6. Canasvieiras.gif é um exemplo de saída, um gif que contém os quarenta anos do distrito, para visualizar as mudanças temporais. Foi feito um para cada um dos 18 distritos
